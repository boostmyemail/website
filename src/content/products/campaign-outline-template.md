I got tired of campaign plans that were too long to read and too vague to act on. So I built a Claude Project that does one thing: takes a planning conversation and outputs a tight, single-page campaign outline in Markdown.

This is the template I use with my own clients. I'm giving it away because the tool is only useful if you actually set it up — and that takes five minutes, not a consulting engagement.

## What's included

### Claude Project instructions

The system prompt that tells Claude how to run the planning session — what to ask, what to pull out of the conversation, and how to structure the output. Drop this into any Claude Project and it works immediately.

### Campaign outline template

The Markdown template that defines the output format: a hypothesis, entry logic, sequence summary, and success metrics. One page. Nothing else.

### Setup guide

A short doc explaining how to wire it together, how to add client-specific context (brand voice, tone, prior campaign history), and how to adapt the output template for different campaign types.

## How it works

1. Create a Claude Project and paste in the system prompt
2. Add any client-specific context files (brand guidelines, tone of voice, account notes)
3. Open the project and talk through the campaign out loud — what you're trying to accomplish, who you're targeting, what the sequence should look like
4. Claude produces the outline. You edit it. It goes to the client.

The output is a starting point. You're still the one signing off on the strategy.
